Summary of Project: Twitch API Metrics

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, data was pulled from the OpenWeatherMap API to aggregate data on over 500 unique cities.

After accumulating the data, Matplotlib was used to plot available variables versus the latitudes of the cities. These variables included temperature, cloudiness, wind speed and humidity. The site provides the source data and visualizations created as part of this analysis, as well as conclusions and insights that were derived from this analysis.
